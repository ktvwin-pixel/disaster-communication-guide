# 재난안전 커뮤니케이션 전략 가이드 배포 안내

## Ⅰ. 즉시 사용

1. `index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
2. 입력값을 넣고 `가이드 생성`을 누르면 체크리스트, 스크립트, 문자메시지, 공개범위가 즉시 갱신됩니다.
3. `PDF/인쇄`를 누른 뒤 대상에서 `PDF로 저장`을 선택하면 Google Drive, Chrome, Google Docs 미리보기에서 열 수 있는 표준 PDF로 저장할 수 있습니다.

## Ⅱ. 모바일 URL 배포

1. `index.html`을 GitHub Pages, Netlify, Cloudflare Pages 같은 HTTPS 정적 호스팅에 업로드합니다.
2. 배포 후 생성된 `https://...` URL을 공유하면 모바일 기기에서도 링크를 가진 사람이 접속할 수 있습니다.
3. 공용 배포 전에는 예시 입력값, 개인정보, 실제 피해자 정보, 내부 연락망이 포함되어 있지 않은지 확인합니다.

## Ⅲ. Google에서 PDF 열기

1. 앱에서 `PDF/인쇄`를 실행합니다.
2. `PDF로 저장`을 선택해 파일명을 지정합니다.
3. 저장된 PDF를 Google Drive에 업로드하면 URL 공유, Drive 미리보기, Chrome 열람이 가능합니다.

## Ⅳ. 주의

1. 이 앱은 정적 HTML 파일이므로 서버에 입력값을 전송하지 않습니다.
2. 브라우저의 `로컬 저장` 기능을 누르면 입력값이 해당 기기의 브라우저 저장소에 남습니다.
3. 실제 사고 정보가 들어간 화면을 공용 URL에 그대로 올리지 마십시오.
4. 법적 판단, 개인정보 공개, 수사·조사 중 사안은 원문 법령과 기관 승인 절차를 우선해야 합니다.
